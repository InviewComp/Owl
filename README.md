![alt text](https://raw.githubusercontent.com/InviewComp/Owl/master/Owl_architecture.jpg)

### Usage:
* Put in `collector/collector.yaml` your OPC-UA server address
* Run `run .sh`

Graph and setting will be available at localhost:31337
